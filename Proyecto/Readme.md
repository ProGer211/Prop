# Proyecto Final – Juego Hex con Inteligencia Artificial

Proyecto desarrollado en la asignatura **PROJECTE DE PROGRAMACIÓ**.

## Descripción

Este proyecto consiste en la implementación de una Inteligencia Artificial para el juego de mesa **Hex**, desarrollada sobre un framework base proporcionado.

El sistema incluye:

- Interfaz gráfica de usuario (GUI)  
- Gestión de interacción entre jugadores  
- Soporte para partidas humano vs IA e IA vs IA  
- Control de tiempo por turno (timeout)  

El objetivo principal fue diseñar e implementar una IA competitiva capaz de tomar decisiones estratégicas dentro de un límite de tiempo establecido.

La IA se basa en:

- Exploración de árboles de juego  
- Funciones heurísticas de evaluación  
- Optimización de profundidad de búsqueda  
- Toma de decisiones eficiente bajo restricciones de tiempo  

## Tecnologías
- Java  
- Swing (interfaz gráfica)  
- Algoritmos de Inteligencia Artificial  
- Búsqueda en árboles de juego
