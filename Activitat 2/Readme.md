# Activitat 2 – Inteligencia Artificial para juego Conecta

Proyecto desarrollado en la asignatura **PROJECTE DE PROGRAMACIÓ**.

## Descripción

El objetivo de esta actividad fue desarrollar una Inteligencia Artificial capaz de vencer a una IA implementada por el profesor en un juego tipo “Conecta”.

La IA fue desarrollada en **Java** y se centra en:

- Algoritmos de búsqueda en árboles de juego  
- Estrategias de toma de decisiones  
- Técnicas de búsqueda adversaria  
- Optimización del rendimiento  

El propósito principal fue diseñar una IA competitiva capaz de analizar el estado del juego y seleccionar movimientos óptimos para derrotar a un oponente avanzado.

## Tecnologías
- Java  
- Algoritmos de Inteligencia Artificial
