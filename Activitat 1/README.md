# Activitat 1 – Ejercicios con Robocode

Proyecto desarrollado en la asignatura **PROJECTE DE PROGRAMACIÓ**.

## Descripción

Esta actividad consta de dos ejercicios basados en **Robocode**, desarrollados en **Java**.

El objetivo fue diseñar e implementar comportamientos inteligentes para robots capaces de competir en batallas simuladas. El proyecto se centra en:

- Programación orientada a objetos en Java  
- Programación basada en eventos  
- Diseño de estrategias de movimiento y ataque  
- Toma de decisiones en entornos competitivos  

Los robots fueron programados para detectar enemigos, reaccionar ante distintos eventos y optimizar su estrategia de movimiento y disparo dentro del entorno de Robocode.

## Tecnologías
- Java  
- Robocode
